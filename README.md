# Notes APP


## Instruction run the application

1. Install react native dependencies 
    ```
    $ yarn
    ```
    
2.  Install iOS dependencies
    ```
    $ cd ios && pod install
    ```
    
3. Run the application
    
    *Start Metro Server*
    ```
    $ yarn start
    ```
    
    **Run in Android emulator**
    
    ```
    $ yarn android
    ```
 
    **Run in iOS simulator**
    
    ```
    $ yarn ios
    ```
